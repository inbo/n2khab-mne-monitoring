- "special" command shortcuts to explore database metadata
- e.g.
	- `\l`    -> list databases
	- `\du`  -> list users
	- `\dn+`  -> list schema's and access rights
	- `\dt+`  -> list tables
	- `\ds+`  -> list sequences
	- `\z "schema"."Tables"` -> table information
	- `\dp "schema"."Tables"` -> permissions
