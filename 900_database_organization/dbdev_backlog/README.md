This sub-folder tracks work and changes on the database.
It is an obsidian vault (*cf.* <https://obsidian.md>) to be used in that software.

- Tasks which are up for grabs are found in the main folder.
- We roughly record start and end date, executing colleague, and status ("done") for filtering purposes.
- These do not replace #issue and #PR! Most tasks are entered retrospectively.
- Tags and properties are used for filtering and searching.
- No formal requirements. Language may vary between Dutch and English.

The use case is to simply open the daily note and log achievements on the database tooling for later reference.


> [!important] These are NOT the [[docs/docs|docs]]!
> This subfolder is an internal process review, and no place for publicly relevant documentation.
> The content of this folder is not guaranteed to be complete or even understandable.


To clarify, we herein document *retrospectively* to keep an archive of fulfilled tasks we might want to find back (preferrably *not*, as we love finished tasks which are really finished).
This is fundamentally different from `990_database_documentation`, a *prospective* documentation folder on #mne-monitoring [^1], which is the main place for #docs and help on recurrent procedures which might also be relevant to others (e.g. future colleagues and collaborators).

[^1]: *cf.* <https://github.com/inbo/n2khab-mne-monitoring/tree/main/990_database_documentation>.
