`precedence_columns` are a *hardcoded list* of database fields which hold user input in the [[locations/MNMDatabaseToolbox.R|MNMDatabaseToolbox.R]] file.
They should be removed; ideally replaced by an extra attribute in the [[locations/structure sheets|structure sheets]].
